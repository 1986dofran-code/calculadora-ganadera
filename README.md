# Calculadora Ganadera

## Descripción
Aplicación web que permite calcular producción y consumo de alimento del ganado.

## Funcionalidades
- Selección de animal
- Tipo de producción
- Total producido
- Cálculo de alimento
- Formulario de contacto
- Diseño responsive

## Pendiente
- Mejorar diseño
- Agregar más cálculos

## Uso
Se ejecuta con Live Server en el navegador.
